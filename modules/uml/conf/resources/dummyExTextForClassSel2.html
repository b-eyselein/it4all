Ein Krankenhaus ist in Stationen, mit einer eindeutigen Nummer aufgeteilt. Jede Stationhat ein oder mehrere Doktoren, Krankenschwestern und Patienten haben jeweils eine eindeutige ID und einen Namen. Ein Doktor kann dem Patienten ein Rezept mit einer eindeutigen Id verschreiben, das ein oder mehrere Medikamente beinhalten kann. Ein Medikament hat eine eindeutige Id und einen Namen. Ein Patient wiederum kann in einer Station aufgenommen bzw. entlassen werden.
