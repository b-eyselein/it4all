Erstellen Sie ein UML-Klassendiagramm, indem Sie die Klassen, inklusive Methoden und Attributen, und die Beziehungen zwischen den Klassen, modellieren.

  • Klassen erzeugen: Mittels Drag-and-Drop können Sie Wörter aus dem Aufgabentext in die Zeichenfläche ziehen, um Klassen zu erzeugen. Alternativ können Sie Klassen auch über den Button "Klasse" erstellen.
  • Klassen verschieben: Mittels gedrückter Maustaste können Sie die jeweilige Klasse verschieben.
  • Elemente löschen: Durch einen Rechtsklick können Elemente gelöscht werden.
  • Bearbeiten einer Klasse (inkl. Attribute und Methoden): Durch einen Doppelklick auf eine Klasse öffnet sich das Bearbeitungsfenster für die jeweilige Klasse. Dort können Sie den Typ und den Namen der Klasse ändern sowie Attribute und Methoden hinzufügen.
