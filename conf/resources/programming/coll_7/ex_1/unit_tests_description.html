Schreiben Sie Unittests für eine Implementierung der Klasse Circle, die einen Kreis mit einem Mittelpunkt (bestehend aus X- und Y-Koordinate und einem Radius) darstellt. Es sollen ein Konstruktor (def __init__(self, center_x: float, center_y: float, radius: float)) und eine Funktion def area(self) -> float: implementiert werden, die die Fläche des Kreises berechnet.
